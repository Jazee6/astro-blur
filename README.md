## 特性

- ✅ 100% 原生组件，纯静态生成
- ✅ Twikoo 评论系统
- ✅ SEO友好 — OpenGraph / Sitemap / RSS
- ✅ 响应式布局 / 代码高亮 / 主题切换

## 快速开始

```shell
#下载模板
npx degit Jazee6/astro-blur#main my-blog

#进入项目
cd my-blog

#安装依赖
pnpm i

#启动项目
pnpm dev

#新建文章 
pnpm new

#配置修改
./src/config.ts
```
